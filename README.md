# Smart-Plant-IoT-Plant-Monitoring-System
Smart Plant is an IoT-based plant monitoring system built with ESP32 that helps prevent plant dehydration by continuously tracking soil moisture, temperature, and humidity in real time.
by continuously tracking soil moisture, temperature, and humidity in real time.

The project was inspired by a simple problem: forgetting to water a plant. Instead of relying on memory, Smart Plant provides live monitoring through a web dashboard and lays the foundation for future automation.

Features
Real-time soil moisture monitoring
Temperature and humidity measurement using DHT11
ESP32 embedded web server
Responsive web dashboard
Moisture history visualization
Manual watering log
Wi-Fi connectivity
Future-ready architecture for automatic irrigation
Hardware
ESP32 Development Board
Capacitive Soil Moisture Sensor
DHT11 Temperature & Humidity Sensor
Power Supply / USB
Software
C++
Arduino Framework
ESP32 WiFi Library
WebServer Library
HTML / CSS / JavaScript Dashboard
Dashboard

The dashboard displays:

Soil moisture percentage
Air temperature
Air humidity
Watering history
Plant health status
Connection status
Future Roadmap
Automatic irrigation using a water pump
Mobile notifications
Cloud data storage
Historical analytics
Multiple plant support
Remote access from anywhere
Motivation

This project started after a plant was unintentionally left without water and nearly died. Instead of replacing the plant, the challenge became an opportunity to build a practical IoT solution that combines embedded systems, sensors, networking, and web technologies.

From a forgotten plant to a smart monitoring system. 🌱🚀
